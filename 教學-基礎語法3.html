陣列

為什麼要用陣列，比較以下程式碼
陣列可以節省很多變數的使用

語法

宣告方式:
  • 資料型態 變數名稱[陣列大小];

配置、輸入、輸出:
  • 配置 : 變數名稱[index] = 改變的值;
  • 輸入 : cin >> 變數名稱[index];
  • 輸出 : cout << 變數名稱[index];

範例1 - 陣列搭配迴圈

字串

標頭檔
char表示字串:
  • 宣告 : char a[字串大小];
  • 輸入 : cin >> 變數名稱; 或 scanf("%s", 變數名稱);
  • 輸出 : cout << 變數名稱; 或 printf("%s", 變數名稱);
  • char字串函式
string表示字串:
  • 宣告 : string a;
  • 輸入 : cin >> 變數名稱; scanf不能用
  • 輸出 : cout << 變數名稱; 或 printf("%s", 變數名稱);
  • string字串函式
